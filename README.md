# CookBook

Recipe discovery app.

## Tech stack

- **Next.js 16** – React framework (App Router)
- **React 19** – UI
- **TypeScript**
- **Tailwind CSS 4** – Styling
- **TanStack Query** – Data fetching & caching
- **Lucide React** – Icons

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
